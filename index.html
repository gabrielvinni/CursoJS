Aprendendo JavaScript

Comandos Basicos

Como toda linguagem de programação, o JavaScript também possui vários comandos, e para começar, irei apresentar os mais simples.
O comando "alert" é utilizado para notificar o usuário sobre algo. Ele desvia o foco da janela atual e obriga o usuário a ler a mensagem, portanto, não abuse muito desse comando, pois isso impede o acesso a outras páginas ou interações do site. Este comando é escrito da seguinte forma:

window.alert("Esse é um alerta")

O próximo comando é o "confirm", e com ele é possível perguntar ao usuário e receber uma resposta, "ok" (sim) ou "cancel" (não). Assim como o "alert", ele também retira o foco da tela principal, esperando a resposta do usuário. Ao contrário do "alert", que não tem um retorno, o "confirm" retorna true ou false, dependendo da escolha do usuário. Este comando é escrito da seguinte forma:

window.confirm("Esse é o confirm, deseja continuar ?")

O comando "prompt", assim como os anteriores, também retira o foco da tela principal, nele o usuário deve inserir algum valor que lhe for sugerido. Este comando é escrito da seguinte forma:

window.prompt("Escreva se nome:")

Podemos guardar esses valores do usuário utilizando as variáveis, que é o tema da próxima sessão.
