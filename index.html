Conhecendo JavaScript

O JavaScript é uma das tecnologias usadas nos sites pela internet, além dele teríamos o HTML e o CSS. O HTML adiciona o conteúdo, o CSS estiliza e o JavaScript cria a interação desse conteúdo com o usuário. Vou dar um pequeno exemplo dessa interação com três comandos simples:

Alert

O comando "alert" é utilizado para notificar o usuário sobre algo. Ele desvia o foco da janela atual e obriga o usuário a ler a mensagem, esse comando é utilizado em casos específicos, pois impede o acesso a outras páginas ou interações do site. Ele é escrito da seguinte forma:

window.alert("Isso é um alerta")
Confirm

Com o confirm é possível perguntar ao usuário e receber uma resposta, "ok" (sim) ou "cancel" (não). Assim como o "alert", ele também retira o foco da tela principal, esperando a resposta do usuário. Ao contrário do "alert", que não tem um retorno, o "confirm" retorna true ou false, dependendo da escolha do usuário. Esse comando é escrito da seguinte forma:

window.confirm("Isso é um confirm, deseja continuar ?")
Prompt

O comando "prompt", assim como os anteriores, também retira o foco da tela principal, mas nele o usuário deve inserir algum valor que lhe for sugerido. Este comando é escrito da seguinte forma:

window.prompt("Escreva seu nome:")

Tirando o alert, os dados que você inseriu no confirm e no prompt se perderam pois não tinham um lugar para serem guardados, na próxima seção vamos ver como guardar esses dados.

Variáveis

As variáveis são como caixas que podem guardar as coisas, e essas coisas seriam os dados. Nós podemos declarar essas variáveis de 4 maneiras:

Declarando automaticamente A = 20
B = "JavaScript"
C = 'p' Esse método não é recomendado pois não declara que é uma variável.
Usando o let let A = 20
let B = "JavaScript"
let C = 'p'
Usando o var var A = 20
var B = "JavaScript"
var C = 'p'
Usando o const const A = 20
const B = "JavaScript"
const C = 'p'
